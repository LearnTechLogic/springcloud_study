# Spring Boot + MyBatis 学生管理系统

## 项目说明
本项目实现了基于 Spring Boot 和 MyBatis-Plus 的学生管理系统，完成了对学生表的增删改查操作。

## 技术栈
- Spring Boot 4.0.5
- MyBatis-Plus 3.4.1
- MySQL 8.0
- Lombok
- Knife4j (API 文档)

## 快速开始

### 1. 数据库初始化
执行 `src/main/resources/sql/init.sql` 脚本创建数据库和学生表：

```sql
-- 创建数据库
CREATE DATABASE IF NOT EXISTS `springboot_mybatis` DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;

USE `springboot_mybatis`;

-- 运行 init.sql 中的建表和插入数据语句
```

### 2. 配置数据库连接
修改 `src/main/resources/application.yaml` 中的数据库连接信息：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/springboot_mybatis
    username: root
    password: 1234
```

### 3. 启动应用
运行 `SpringbootMybatisApplication` 主类启动应用。

### 4. 访问 API 接口
应用启动后，可以通过以下方式访问：

#### API 文档地址
- Knife4j 文档：http://localhost:8080/doc.html

#### 学生管理接口
基础路径：`/api/students`

| 方法 | 路径 | 描述 |
|------|------|------|
| GET | /api/students | 查询所有学生 |
| GET | /api/students/page?pageNum=1&pageSize=10 | 分页查询学生 |
| GET | /api/students/{id} | 根据 ID 查询学生 |
| POST | /api/students | 添加学生 |
| PUT | /api/students | 更新学生信息 |
| DELETE | /api/students/{id} | 删除学生 |
| DELETE | /api/students/batch?ids=1,2,3 | 批量删除学生 |
| GET | /api/students/condition?name=张&className=计算机 1 班 | 根据条件查询学生 |

### 5. 测试示例

#### 添加学生
```bash
curl -X POST http://localhost:8080/api/students \
  -H "Content-Type: application/json" \
  -d '{
    "studentNo": "2024008",
    "name": "测试学生",
    "age": 20,
    "gender": "男",
    "className": "计算机 3 班"
  }'
```

#### 查询所有学生
```bash
curl http://localhost:8080/api/students
```

#### 更新学生信息
```bash
curl -X PUT http://localhost:8080/api/students \
  -H "Content-Type: application/json" \
  -d '{
    "id": 1,
    "studentNo": "2024001",
    "name": "张三丰",
    "age": 25,
    "gender": "男",
    "className": "计算机 1 班"
  }'
```

#### 删除学生
```bash
curl -X DELETE http://localhost:8080/api/students/1
```

## 功能说明

### 1. 实体类（Student）
- id: 主键 ID（自增）
- studentNo: 学号
- name: 姓名
- age: 年龄
- gender: 性别
- className: 班级
- createTime: 创建时间
- updateTime: 更新时间

### 2. 主要功能
- ✅ 查询所有学生
- ✅ 分页查询学生
- ✅ 根据 ID 查询学生
- ✅ 添加学生
- ✅ 更新学生信息
- ✅ 删除学生（单个/批量）
- ✅ 根据条件查询学生（姓名/班级）

### 3. MyBatis-Plus 特性
- 使用 BaseMapper 提供的基础 CRUD 方法
- 使用 QueryWrapper 进行条件查询
- 使用 Page 进行分页查询
- 自动填充创建时间和更新时间
- 驼峰命名与下划线命名自动转换

## 注意事项
1. 确保 MySQL 数据库已启动并可正常访问
2. 数据库用户名和密码需要根据实际情况修改
3. 端口号默认为 8080，如有冲突可在 application.yaml 中修改
